For Application Administrators

1. Check application logs:

  • Look for log files in log/production.log or your logging service
  • The most recent error should be at the bottom of the log file

2. Recent Exchange Rate Changes:

  • If you recently modified exchange rate handling code, check if all exchange rates are available
  • Verify SQL joins on the exchange_rates table
  • Look for issues with the LATERAL join syntax in SQL queries

3. For Development:

  • Set config.consider_all_requests_local = true in config/environments/production.rb temporarily to see detailed error pages
  • Or copy the error from logs and debug locally in development environment

Technical note: This error may be related to missing exchange rates or SQL LATERAL join syntax in PostgreSQL. Check the version compatibility of your database.
